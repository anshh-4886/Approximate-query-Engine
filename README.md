<h1 align="center">🚀 Approximate Query Processing Engine</h1>

<p align="center">
  A professional interactive analytics platform for executing large-scale queries
  <strong>faster than exact computation</strong> while maintaining high accuracy.
</p>

<p align="center">
  Built for hackathon use cases involving <strong>streaming data</strong>,
  <strong>skewed datasets</strong>, and <strong>high-cardinality data</strong>.
</p>

<hr>

<h2>📌 Overview</h2>
<p>
  The <strong>Approximate Query Processing Engine (AQP)</strong> is designed to answer
  analytical queries efficiently by using multiple optimized strategies instead of
  scanning the full dataset every time. It supports both static and live-streaming
  datasets, allowing users to compare <strong>exact vs approximate execution</strong>
  in terms of <strong>speed, accuracy, and scalability</strong>.
</p>

<h2>✨ Features</h2>
<ul>
  <li>⚡ Fast query execution with approximate processing techniques</li>
  <li>📊 Supports <strong>COUNT</strong>, <strong>SUM</strong>, <strong>AVG</strong>, and <strong>GROUP BY</strong></li>
  <li>📡 Live streaming simulation support</li>
  <li>🧠 Multiple query strategies for different data behaviors</li>
  <li>📈 Accuracy vs speed comparison dashboard</li>
  <li>📂 CSV upload support for custom datasets</li>
  <li>📉 Benchmark logging and downloadable performance reports</li>
  <li>🎨 Premium interactive Streamlit UI</li>
</ul>

<h2>🧠 Query Processing Strategies</h2>
<ul>
  <li><strong>Uniform Sampling</strong> – Best for evenly distributed datasets</li>
  <li><strong>Stratified Sampling</strong> – Better for skewed data distributions</li>
  <li><strong>Pre-Aggregated Synopsis</strong> – High-accuracy strategy for repeated analytical queries</li>
  <li><strong>Streaming Incremental Aggregation</strong> – Optimized for live and continuously incoming data</li>
  <li><strong>Hash Bucket Summaries</strong> – Suitable for high-cardinality and near-unique values</li>
  <li><strong>Sketch-based Estimation (Count-Min Sketch)</strong> – Fast probabilistic counting for large-scale data</li>
</ul>

<h2>🎯 Project Goal</h2>
<p>
  The goal of this project is to show how modern data systems can balance
  <strong>accuracy</strong> and <strong>performance</strong> using intelligent approximation methods.
  Instead of relying on a single technique, this engine uses a
  <strong>hybrid multi-strategy approach</strong> so that different kinds of datasets
  can be handled more effectively.
</p>
